# G-3DS 
Garry's Mod for the 3DS! Made with Octave!

## WHAT IS G-3DS?
<img src=https://file.garden/aQ9-6gw2fD_KMuI9/Screenshot.png width=300>

G-3DS is a Garry's Mod **DEMAKE** for the 3DS!

In Version 1, it includes 8 ITEMS:
* A cardboard box
* An explodable Barrel (Explodes if you shoot it!)
* A toy car (That you can ride with SELECT)
* A bouncing ball
* A potion (That when thrown at props, makes them fly for 5 seconds!)
* A frog chair
* A lamp (To illuminate your nightly actions!)
* And SUPA MARIO (He deletes all props!)

## CONTROLS
* CirclePad to look
* ABXY to walk
* R to jump
* L to use the weapons
* DPAD Down when holding items with the Physics Gun to throw them!
